言語は英語・簡体中国語に対応。適当な翻訳ではあるので何卒。<br>

## 新規・追加・変更機能
### 霊界での視野拡大縮小(MODクライアント限定)/TOH_Yでの追加
霊界でマウスホイールを動かすと、マップ全体が見えるようになったりします。<br>
MOD入りインポスターが千里眼を使えるかどうか設定できます。<br>

### /w コマンドの追加
ホストのみ使用可能。
「/w (impostor/crewmate/none/jackal)」を入力することで、その陣営の勝利としてその試合を終了できる。

### /messagewait or /mw <秒> コマンド
小数に対応できるように変更。<br>

### F11の画面サイズ
ゆめのがいつも使っている1366×768を含めるなど少々変更。

### 幽霊が他人の役職・投票先を見ることができる設定
「幽霊が他人の役職・投票先を見ることができる」
↓
「幽霊が他人の役職・投票先を見ることができない」

### 役職設定画面の新レイアウト
各陣営ごとのタブ、madmate settingはYのみ。
アイコンはとりにく氏提供。

### /nで表示される設定
キルフラッシュの長さ・自動的に試合結果を表示・名前の二行目・役職割り当てのアルゴリズムを表示しないように変更。

## TOH_Y追加役職

### イビル猫又

陣営 : インポスター<br>
判定 : インポスター<br>

吊られた時に1人をランダムに道連れ(吊られた次の議論開始時に死体演出無しで×マークが付く)<br>
インポスターを道連れにすることは絶対にない。<br>

#### 設定
| 設定名                              |
| ----------------------------------- |
| マッド系役職を道連れできる            |


### 猫又

陣営 : クルーメイト<br>
判定 : クルーメイト<br>

吊られた時に1人をランダムに道連れ(吊られた次の議論開始時に死体演出無しで×マークが付く)<br>

### マッドメイト系役職のオプション追加

吊られた時に1人をランダムに道連れ(吊られた次の議論開始時に死体演出無しで×マークが付く)<br>
インポスターを道連れにすることは絶対にない。<br>

#### 設定
| 設定名                              |
| ----------------------------------- |
| マッド系役職追放時、クルーを道連れ      |

### 猫又系役職のオプション

イビル猫又、猫又、マッド系役職道連れオンに関する。<br>

#### 設定
| 設定名                              |
| ----------------------------------- |
| 猫又系役職の道連れ連鎖                |
| 第三陣営を道連れできる                |

> **Warning**  
>シェリフが道連れされた時、20秒ほど暗転する現象を確認。<br>

### アンチアドミナー

陣営 : インポスター<br>
判定 : インポスター<br>

アドミン・ドアログ・バイタル機器前に人がいるか分かる。実際に機器を使用しているかは分からない。<br>

#### 設定
| 設定名                              |
| ----------------------------------- |
| カメラ前にいる人も分かる             |

### 呪狼

陣営 : インポスター<br>
判定 : インポスター<br>

誰かにキルされそうになった時、その人からキルをガードして呪殺する。<br>

#### 設定
| 設定名                              |
| ----------------------------------- |
| キルを防いで呪殺する回数             |

### マッドディクテーター

陣営 : インポスター<br>
判定 : クルーメイトorエンジニア<br>
カウント : クルー<br>

クルーだがインポスターの味方をする。インポスターと互いに認識できない。<br>
会議中に誰かに投票をすると、会議を強制終了させて投票先を吊る事ができる。投票したタイミングで自身も死ぬ。<br>
クルーの[ディクテーター](#Dictatorディクテーター)と仕様は同じ。<br>

#### 設定

| 設定名                       |
| ---------------------------- |
| ベントを使える               |

### マッドネイチャコール

陣営 : インポスター<br>
判定 : エンジニア<br>
カウント : クルー<br>

クルーだがインポスターの味方をする。インポスターと互いに認識できない。<br>
エアシップの時のみ効果を発動する。ベントに入る度ラウンジトイレのドアが4つ同時に開く 。<br>

### マッドブラックアウター

陣営 : インポスター<br>
判定 : エンジニア<br>
カウント : クルー<br>

クルーだがインポスターの味方をする。インポスターと互いに認識できない。<br>
ベントに入ることで停電を起こすことが出来る。（ベントクールダウンを長めに設定するべし）<br>
船員はこの停電をいつも通り直すことが出来る。<br>
自身が起こした停電中にもう一回ベントに入ると停電は解除される。<br>
インポスターが起こしたサボには重ねることが可能（コミュサボ中はそもそもベント入れないが）。<br>

### マッドシェリフ

陣営 : インポスター<br>
判定 : インポスター<br>
カウント : クルー<br>

クルーだがインポスターの味方をする。インポスターと互いに認識できない。<br>
相手がどの陣営に関わらず、誰をキルしようとしても必ず自爆する。<br>

#### 設定
| 設定名                              |
| ----------------------------------- |
| キルクール                            |
| ベント使用（ベントクールダウンは設定不可）|
| ターゲットも同時に死ぬ                  |

### ハンター

陣営 : クルーメイト<br>
判定 : インポスター<br>

誰に向かってでもキルできる。()内回数キル可能。<br>
あくまでクルー陣営なのでタイミングは考えよう。タスクはもっていない。<br>

#### 設定
| 設定名                              |
| ----------------------------------- |
| キルクール                          |
| キル可能回数                       |
| インポスター視界                   |

### タスマネ

陣営 : クルーメイト<br>
判定 : クルーメイト<br>

タスクマネージャー。<br>
生存時、会議中に全員合計のタスク完了数が分かる。<br>

#### 設定
| 設定名                              |
| ----------------------------------- |
| 会議時以外でもタスク数が見える        |

### パン屋

陣営 : クルーメイト<br>
判定 : クルーメイト<br>

生存中、会議はじめにパン屋についてのコメントが流れる。レアコメントあり。<br>

### エクスプレス

陣営 : クルーメイト<br>
判定 : クルーメイト<br>

スピードが他の人より速いだけの確白。必ずx3になる。<br>

### チェアマン

陣営 : クルーメイト<br>
判定 : エンジニア<br>

・ベントに入ると会議が開ける。サボタージュ中はベントに入っても会議を開けない。<br>
通常ボタン回数と同時に消費する。(通常ボタン回数1回の時、先にベントボタンを使うと、通常ボタンは使えなくなる。)<br>
・誰かに投票することでその会議を即中断できる(追放は起こらずスキップ扱い)。<br>
スキップ・自投票以外の普通の投票は出来ず、能力を発動してしまう。<br>

| 設定名                              |
| ----------------------------------- |
| ボタンの使用可能回数(通常ボタン回数も含む)|

### にじいろスター

陣営 : クルーメイト<br>
判定 : クルーメイト<br>

自分の役職名がカラフル。体の色が1秒ごとに変わる。全員視点からにじいろスターが誰か分かる。<br>
自身のタスクカウントは見えない。<br>

### 見送り人

陣営 : クルーメイト<br>
判定 : クルーメイト<br>

追放確認が自身だけオンになる。追放した人がインポスターかそうでないかがわかる。<br>

> **Warning**  
> インポスターが1の時、暗転対策として必ず同数投票と表示される仕様により、見送り人は使用不可となる。

### バカシェリフ

陣営 : クルーメイト<br>
判定 : インポスター<br>

普通のシェリフとほとんど変わりないが、正しく陣営を認識できない時がある。<br>
(人外に向かって自爆・クルーをキルする時がある)<br>

| 設定名                              |
| ----------------------------------- |
| 通常のシェリフの設定全て              |
| 正しく認識しない確率                 |

### 共鳴者

陣営 : クルーメイト<br>
判定 : クルーメイト<br>

共鳴者は必ず2人でしか設定できない。<br>
設定タスク数まで完了させると自身にしか見えない◎がつく。2人ともタスクが互いに設定数を超えた時、相手にも◎が付きもう1人の共鳴者が誰か分かる。<br>

| 設定名                              |
| ----------------------------------- |
| 相方が確認できるタスク数              |

### アンチコンプリート

陣営 : 第三（単独）<br>
判定 : クルーメイト<br>
カウント : クルー<br>
勝利条件 : 会議中、全体で1回のみ好きな人を投票して、その人がタスクを既に完了させていること。<br>

タスクがない人、タスクが終わってない人を選ぶと死亡し敗北。<br>
選択は[ディクテーター](#Dictatorディクテーター)方式だが、選択された人は死亡しない。<br>
追放時の演出は、アンチコンプリート自身が表示される。<br>

### ワーカホリック

陣営 : 第三（単独）<br>
判定 : エンジニア<br>
カウント : クルー<br>
勝利条件 : タスクを終わらすこと。<br>

タスク量はとても多く設定するべきもの。ベント可能。<br>
死亡しても関係なく、タスクが終わると勝利。霊界からでも終わらそう。<br>

#### 設定

| 設定名                              |
| ----------------------------------- |
| タスク数                            |
| ベントクールダウン                   |
| 全員視点からワーカホリックがだれか分かる     |
| ┗全員視点からワーカホリックのタスク数が分かる |

### ダークハイド

陣営 : 第三（単独）<br>
判定 : インポスター<br>
カウント : クルー<br>
勝利条件 : ・自身生存時にインポスター陣営が勝利した時　・自身生存時にインポスター(設定:+斬れる第三陣営)を1人以上斬ってクルー陣営が勝利(タスク勝利除く)した時<br>

誰でも切れる。ベント不可。キルを起こす度に停電が起こる。(既に停電の場合はそのまま)。<br>

#### 設定

| 設定名                              |
| ----------------------------------- |
| キルクール                          |
| インポスター視界                   |
| キル可能第三陣営もカウントに含むか|

### ラブカッター

陣営 : 第三（単独）<br>
判定 : クルーメイト<br>
カウント : クルー<br>
勝利条件 : 設定回数誰かにキルされたら単独勝利。<br>

ラブカッターがキルされた時、キルをガードしてカウントする。
基本キル役職と同数回以下に設定するべきもの。

#### 設定

| ----------------------------------- |

## 既存TOH役職＿TOH_Yによる仕様変更

#### シェリフ
キルクールの設定が1秒間隔 → 2.5秒間隔に変更。<br>

#### シュレディンガーの猫
色変化関連の変更。<br>
シェリフに切られた時、シュレ猫視点シェリフがシェリフ色(黄色)に変化。（追加されたハンターも同様。）<br>
インポスターに切られた時、シュレ猫視点、【全員のインポスター】の色がインポスター色(赤色)に変化。<br>

#### GM
キルフラッシュが見えるように変更。<br>

#### 恋人
・日本語訳を変更(恋人→ラバーズ)<br>
・♡マークを塗りつぶしハートに変更<br>

#### オポチュニスト
キルできる設定を追加。ベント、サボタージュ不可。<br>
オポチュニストキラーに斬られたシュレディンガーの猫は、オポチュキラーの生存の有無に関わらず、自身が生き残っていれば追加勝利となる。<br>
