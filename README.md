## ★新規・追加・変更機能
### 霊界での視野拡大縮小(MODクライアント限定)/TOH_Yでの追加
霊界でマウスホイールを動かすと、マップ全体が見えるようになったりします。<br>
MOD入りインポスターが千里眼を使えるかどうか設定できます。<br>

### /w コマンドの追加
ホストのみ使用可能。<br>
「/w (impostor/crewmate/none/jackal)」を入力することで、その陣営の勝利としてその試合を終了できる。<br>

### /messagewait or /mw <秒> コマンド
小数に対応できるように変更。<br>

### F11の画面サイズ
ゆめのがいつも使っている1366×768を含めるなど少々変更。<br>

### 幽霊が他人の役職・投票先を見ることができる設定
「幽霊が他人の役職・投票先を見ることができる」<br>
↓<br>
「幽霊が他人の役職・投票先を見ることができない」<br>

### 役職設定画面の新レイアウト
各陣営ごとのタブ、madmate settingはYのみ。<br>
アイコンはとりにく氏提供。<br>

### /nで表示される設定
キルフラッシュの長さ・自動的に試合結果を表示・名前の二行目・役職割り当てのアルゴリズムを表示しないように変更。<br>

### 道連れ通知
道連れが起こった時、次の会議時に道連れされた人を表示させる設定を追加。<br>

### 会議が開かれた
ボタンかレポートか表示する設定を追加。死体レポートの時、誰の死体が通報されたかも表示される。<br>

### 役職説明の自動表示
一回目の会議時に、自身の役職説明を自動で表示する機能を追加。<br>

### 複数属性の省略表示
2つ以上の属性を持った時、自身の役職名の前に＋が付く。設定で、省略せずに表示させることができる。<br>

### イントロ変更(MODクライアントのみ)
マッドメイト系役職・第三陣営のイントロ表示を変更。<br>

マッドメイト系役職：初めからマッドメイトと表示されず、クルーと表示されてからマッドメイト(詳細役職表示)になる。<br>
第三陣営：ジャッカル以外の単独第三陣営は、初めに第三陣営と表示されてから詳細役職表示になる。<br>
本家TOH仕様に戻す設定あり。 <br>

### 役職設定モード
旧設定モードを、ON-OFF設定モードとし、0%-100%表示をオンオフに変更。<br>
また、属性のみモードを新規追加。現在各バニラ役職につき3パターンまで設定可能。ランダム付与も設定可能。<br>
確率設定モードはv10で実装予定。<br>
※スタンダードゲームモードのみ<br>

## ★TOH_Yオリジナルゲームモード【猫取合戦】
### 概要
ここはそれぞれ孤独な猫たちが集う場。いつも自身のタスクをこなしながら穏やかな日々を過ごしていた。<br>
或る日、この地の神がここの孤独な猫たちを見て、裏切り者が生まれた時に備えていくつか集団を作るべきと判断し、数名にリーダーを任命した。<br>
[紹介動画](https://youtu.be/oWz-rTnxwS0)<br>

### 役職
[リーダー]赤 1/青 1(8人以上推奨/黄 1)<br>
[無陣営]猫 その他<br>
> **info**  
> v411.7から、インポスターの人数を強制的に1に設定されるようになりました。

### リーダーのルール
・各リーダーは、無陣営の猫にキルボタンを押すことで、自陣営の仲間に出来る。<br>
・リーダー同士は、互いにキルができるかどうか設定で決める。<br>
・各リーダー視点、自陣営にした猫は自身と同じ色で表示される。<br>
・どこかの陣営に就いた猫をキルできるかは設定で決める。キルできる設定の時、自陣営の猫であってもキルできるので注意。<br>
・リーダーの視界はインポスタービジョンで設定する。<br>

### 猫のルール
・猫は初めてキルされた時、キルを防いでそのキルをしてきた陣営に所属する。2回目以降キルされるかは設定で決める。<br>
・猫の視界は、クルービジョンで設定する。<br>
・タスクは出来るが関係ない。<br>

### 決着は以下の2通り
(1)無陣営猫が居なくなった時、生存者の中で一番自陣営が多かった陣営が勝利。<br>
(2)リーダーをキルできる設定がオンの時で、他陣営のキルリーダーが全員死んでいた時、残ったキルリーダーの陣営の勝利。<br>

### その他
[会議・死体通報]会議を開くと、毎回生存中の無陣営人数、リーダー生存数を知ることが出来ます。<br>
[キルクール]無陣営の猫を仲間にする時や、対象者をキルできない設定がオンの時のキルガードが発生した時、キルクールは設定時間の半分になる。会議後や、誰かをキルした時は、設定時間通りのキルクールになる。<br>

### 禁止事項
・会議時の追放は出来ない。どこに入れても絶対スキップになる。<br>
・ドア閉め含むサボタージュ禁止。<br>
・ベントはリーダーのみ使えるか設定可能。<br>

設定)

| ----------------------------------- |

| リーダー同士はキルできない           |

## ★TOH_Yオリジナルゲームモード【ワンナイト】
### 勝利条件
(人狼陣営)人狼が誰も死亡しなければ勝利。<br>
(村陣営)人狼を1人でも追放できれば勝利。<br>
(吊人)会議で追放、または狩人に処刑されたら勝利。<br>

### 試合の流れ
（１）行動ターン。村陣営は死体捜索、人狼はバレないようにキルしよう。<br>
・行動ターンの会議ボタンは押せない。死体発見で会議をおこそう。<br>
・村陣営はタスク勝利出来ない。<br>
・タスクを持つ役職は、タスクを全て完了させることで死体がある位置への矢印が表示される。<br>
・人狼(大狼)は、全員1人1キル完了するとドア閉め以外のサボタージュも使用可能になる。<br>
・キルボタンを持つ役職は、はじめにバリアが張られるが無視でOK。(キルクール調整)<br>

（２）死体発見で会議ターン開始。各陣営、勝利条件に向かって話し合い。<br>
・同数投票の時、その同数全員吊り対象になる。<br>
・スキップは基本票数が無視される。全員無投票、全員スキップの時のみ、誰も追放されずにゲームが終了する。＝人狼陣営勝利<br>

### 役職説明
【人狼】1回キルしたらその後キルできなくなる特殊仕様。（タスク不可）<br>
【大狼】キルした人の役職がわかる人狼。会議までにキルしなかった場合、死亡した人からランダムで1人の役職を知ることができる。（タスク不可）<br>
【狂人】人狼を助けよう。誰が人狼かは分からない。<br>
【狂信者】タスク完了で、死体場所がわかる他に人狼が誰かわかる狂人。<br>

【村人】タスクを全て完了して、死体場所を探そう。特殊能力はない。<br>
【占い師】タスクターン中、キルボタンを使用して対象者の役職を確認できる。<br>
会議までに使用しなかった場合、死亡した人からランダムで1人の役職を知ることができる。（タスク不可）<br>
占いは必ず怪盗交換前の役職がわかる。（例）人狼と交換した怪盗を占うと怪盗と表示、怪盗と交換された狩人を占うと狩人と表示。<br>
【怪盗】キルボタンを使用して、対象者の役職と自身の役職を交換することが出来る。<br>
交換された人は自身と交換されたという事を認知しない。<br>
人狼と交換した時、相方がいる場合でも誰が相方か分からない。<br>
タスクターン中は、交換した役職の能力を使用することは出来ない。(タスク不可)
【村長】会議の投票時、1人で2票同じ人のところに投票できる。<br>
【狩人】自身が追放された時、再び狩人投票用の会議が開かれる。<br>
ここでは追放されたら狩人以外は議論するべからず。1人を投票で選択して道連れ処刑することができる。<br>
【パン屋】会議ターン中に生存している時、誰かにパン屋を届ける。誰か一人にしか届けられない。<br>
【罠師】キルされた時、キルした人をその場で足止めさせる。<br>

【吊人】会議で追放されたら勝利。狩人の処刑も含む。 <br>

タスク完了しても会議中はスニッチ視点、キル役職が分からない設定を追加。<br>
また、キル役職に通知が届くタスク残数の設定を追加。 <br>

#### ベイト
ベイトキルから通報までの時間を決める設定を追加。<br>
キラーがベイトをキルした時、自身の名前に！マークが付く。<br>

#### ライター
効果が発動されるタスク数設定を追加。<br>

#### シェリフ
「アドミン・カメラ等の機器が使用できない」「停電・通信妨害が直せない」オプションを追加。<br>

#### シュレディンガーの猫
色変化関連の変更。また、役職名にチーム名を表示するように変更。<br>
シェリフに切られた時、シュレ猫視点シェリフがシェリフ色(黄色)に変化。（追加されたハンターも同様。）<br>
インポスターに切られた時、シュレ猫視点、【全員のインポスター】の色がインポスター色(赤色)に変化。<br>

#### 恋人
・日本語訳を変更(恋人→ラバーズ)<br>
・♡マークを塗りつぶしハートに変更<br>
・追加勝利できる設定を追加。<br>

#### オポチュニスト
キルできる設定を追加。ベント、サボタージュ不可。<br>
キル回数制限の設定を追加。<br>
オポチュニストキラーに斬られたシュレディンガーの猫は、オポチュキラーの生存の有無に関わらず、自身が生き残っていれば追加勝利となる。<br>

#### エクスキューショナー
クルーに役が変わる時のみタスクがある仕様を変更。<br>
クルーに変更される場合でもタスクはしなくてよい。<br>

#### ラストインポスター
属性を付与できる設定を追加。それにより全インポスター役職、ラストインポスターになれるように変更。<br>
キルクールは変更するかどうか設定可能に変更。キルクールに関しては、以前と同じく変更設定不可な役職がある。<br>

不可な役職：ヴァンパイア/バウンティハンター/シリアルキラー/グリーディア/アンビショナー<br>

2) 弁護士が生存していたら、勝利チームの勝利を横取りする(弁護士の単独勝利)。<br>

弁護士は、依頼人となる インポスターorキル可能な第3陣営 を擁護する。<br>

弁護士、追跡者共にタスクはもたない。<br>

依頼人は会議中の助けを得る為に、しばらくの間は弁護士を生かしておく必要があるが、勝利を盗まれない様、適切な時に弁護士を排除しなければならない。<br>

「依頼人は弁護されていることがわかる」がオンの場合、依頼人は自分が弁護される事を認識する(自身にマークが付く)が、誰が弁護士なのかは分からない。弁護士が死亡するとマークは消える。<br>

ジャッカル陣営のマッドスニッチ。
ジャッカル陣営に属しますが、クライアントとジャッカルはお互い誰かはわかりません。
タスクを全て完了させるとジャッカルを認識できるようになります。

後追いをオンの場合、クライアントはタスク完了後、ジャッカルの死を会議で確認した後に死亡する。

#### 設定

| 設定名                              |
| ----------------------------------- |
| ベントを使用できる           |
| ┗ベントクールダウン       |
| ┗ベント内での最大時間   |
| ジャッカルからも視認できる        |
| ジャッカル死亡後のモード       |
| ┗なし or 後追い        |
| 属性を付与する        |

### トトカルチョ

陣営 : 第三（単独）<br>
判定 : インポスター<br>
カウント : クルー<br>
勝利条件 : 賭けた対象プレイヤーが勝利すると、無条件で自身も追加で勝利。<br>

キルボタンを使用して、誰かに「賭ける」ことが可能。<br>
「賭ける」先は設定により途中で変更することも可能。ただし生存人数が減るごとに「賭ける」クールタイムが長くなる。<br>

#### 設定

| 設定名                              |
| ----------------------------------- |
| 「賭ける」の初期クールタイム            |
| 「賭ける」の変更できる回数            |
| 「賭ける」の最終クールタイム            |

### ウォッチング
・属性<br>
匿名投票がオフになり、誰がどこに投票しているか分かるようになる。<br>

### ライティング
・属性<br>
視界が広くなる。設定で、停電中でも変わらない視界で見ることができる。<br>

| 設定名                              |
| ----------------------------------- |
| クルー視界増加率             |
| インポスター視界増加率             |
| 停電を無効にする             |

### サングラス
・属性<br>
視界が狭くなる。もし減少率により、0を下回ると見える部分がなく真っ暗になるので注意。<br>

| 設定名                              |
| ----------------------------------- |
| クルー視界減少率            |
| インポスター視界減少率            |

### シーイング
・属性<br>
プレイヤーが死ぬ時、キルフラッシュが見える。<br>

### オートプシー
・属性<br>
会議時に、死亡したプレイヤーの死因を確認することが出来る。<br>

### VIP
・属性<br>
自身がキルされた時、全員にキルフラッシュが起こる。<br>

### クラムシー
・属性<br>
停電、通信妨害を直すことが出来ない。<br>

### リベンジャー
・属性<br>
自身が会議で追放された時、ランダムなプレイヤーを道連れする。<br>
自身がインポスター・マッドメイトの時、インポスターは道連れされない。<br>

### マネジメント
・属性<br>
会議中、全員の完了タスク合計数がわかる。<br>

| 設定名                              |
| ----------------------------------- |
| タスクターン中にもわかる            |

### インフォプアー
・属性<br>
アドミン・カメラ・ドアログ・バイタルが使用できない。<br>

### タイブレーカー
・属性<br>
もし同数投票があった時、タイブレーカーがそのどれかに入れていた方が優先されて同数投票を壊す。<br>
タイブレーカーが2人以上いてそれぞれ違う同数の所に入れている場合は同数になる。<br>

### ノンレポート
・属性<br>
死体通報ができない。<br>

### センディング
・属性<br>
追放された人がインポスターかそうでないか分かる。<br>

### ロイヤルティ
・属性<br>
※マッドスニッチを除くマッド役職のみ追加で設定可能。<br>
同陣営の親が分かる(マッドメイト→インポスター)。<br>

### プラスボート
・属性<br>
会議の投票時、1人で2票同じ人のところに投票できる。<br>

### ガーディング
・属性<br>
1回無条件でキルを防ぐことが出来る。自身への守ったバリアは見えないので知らない間に消費されているかも。<br>

### ベイティング
・属性<br>
自身をキルしてきた人にセルフレポートさせる。<br>

### リフュージング
・属性<br>
会議で追放されそうになった時、一度拒んで生き残ることができる。<br>

### コンプリートクルー
・属性<br>
タスクを全て完了させた人に、早い順に設定人数分まで付与される。<br>
設定された属性を持つようになる。<br>
