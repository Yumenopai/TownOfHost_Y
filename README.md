## ★TOH_Yオリジナルゲームモード【猫取合戦】
### 概要
ここはそれぞれ孤独な猫たちが集う場。いつも自身のタスクをこなしながら穏やかな日々を過ごしていた。<br>
或る日、この地の神がここの孤独な猫たちを見て、裏切り者が生まれた時に備えていくつか集団を作るべきと判断し、数名にリーダーを任命した。<br>
[紹介動画](https://youtu.be/oWz-rTnxwS0)<br>

### 役職
[リーダー]赤 1/青 1(8人以上推奨/黄 1)<br>
[無陣営]猫 その他<br>
> **Warning**  
> リーダーの数に関わらず、インポスターの人数は1に設定してください。

### リーダーのルール
・各リーダーは、無陣営の猫にキルボタンを押すことで、自陣営の仲間に出来る。<br>
・リーダー同士は、互いにキルができるかどうか設定で決める。<br>
・各リーダー視点、自陣営にした猫は自身と同じ色で表示される。<br>
・どこかの陣営に就いた猫をキルできるかは設定で決める。キルできる設定の時、自陣営の猫であってもキルできるので注意。<br>
・リーダーの視界はインポスタービジョンで設定する。<br>

### 猫のルール
・猫は初めてキルされた時、キルを防いでそのキルをしてきた陣営に所属する。2回目以降キルされるかは設定で決める。<br>
・猫の視界は、クルービジョンで設定する。<br>
・タスクは出来るが関係ない。<br>

### 決着は以下の2通り
(1)無陣営猫が居なくなった時、生存者の中で一番自陣営が多かった陣営が勝利。<br>
(2)リーダーをキルできる設定がオンの時で、他陣営のキルリーダーが全員死んでいた時、残ったキルリーダーの陣営の勝利。<br>

### その他
[会議・死体通報]会議を開くと、毎回生存中の無陣営人数、リーダー生存数を知ることが出来ます。<br>
[キルクール]無陣営の猫を仲間にする時や、対象者をキルできない設定がオンの時のキルガードが発生した時、キルクールは設定時間の半分になる。会議後や、誰かをキルした時は、設定時間通りのキルクールになる。<br>

### 禁止事項
・会議時の追放は出来ない。どこに入れても絶対スキップになる。<br>
・ドア閉め含むサボタージュ禁止。<br>
・ベントはリーダーのみ使えるか設定可能。<br>

## ★既存TOH役職＿TOH_Yによる仕様変更・追加機能
#### 通常インポスター
役職設定画面に「通常インポスター」を追加。<br>
ただ名前に通常がついたもの。今まで通り設定しなくても使えるし、人数・役職把握をしやすくするために設定してもよい。<br>

#### シェリフ
キルクールの設定が1秒間隔 → 2.5秒間隔に変更。<br>

#### ドクター
バイタルを持たない設定を追加。<br>

#### シュレディンガーの猫
色変化関連の変更。<br>
シェリフに切られた時、シュレ猫視点シェリフがシェリフ色(黄色)に変化。（追加されたハンターも同様。）<br>
インポスターに切られた時、シュレ猫視点、【全員のインポスター】の色がインポスター色(赤色)に変化。<br>

#### 恋人
・日本語訳を変更(恋人→ラバーズ)<br>
・♡マークを塗りつぶしハートに変更<br>
・追加勝利できる設定を追加。<br>

#### オポチュニスト
キルできる設定を追加。ベント、サボタージュ不可。<br>
オポチュニストキラーに斬られたシュレディンガーの猫は、オポチュキラーの生存の有無に関わらず、自身が生き残っていれば追加勝利となる。<br>

#### エクスキューショナー
クルーに役が変わる時のみタスクがある仕様を変更。<br>
クルーに変更される場合でもタスクはしなくてよい。<br>

### アンチコンプリート

陣営 : 第三（単独）<br>
判定 : クルーメイト<br>
カウント : クルー<br>
勝利条件 : 会議中、全体で1回のみ好きな人を投票して、その人がタスクを既に完了させていること。<br>

タスクがない人、タスクが終わってない人を選ぶと死亡し敗北。<br>
選択は[ディクテーター](#Dictatorディクテーター)方式だが、選択された人は死亡しない。<br>
追放時の演出は、アンチコンプリート自身が表示される。<br>

### ワーカホリック

陣営 : 第三（単独）<br>
判定 : エンジニア<br>
カウント : クルー<br>
勝利条件 : タスクを終わらすこと。<br>

タスク量はとても多く設定するべきもの。ベント可能。<br>
死亡しても関係なく、タスクが終わると勝利。霊界からでも終わらそう。<br>

#### 設定

| 設定名                              |
| ----------------------------------- |
| タスク数                            |
| ベントクールダウン                   |
| 全員視点からワーカホリックがだれか分かる     |
| ┗全員視点からワーカホリックのタスク数が分かる |

### ダークハイド

陣営 : 第三（単独）<br>
判定 : インポスター<br>
カウント : クルー<br>
勝利条件 : ・自身生存時にインポスター陣営が勝利した時　・自身生存時にインポスター(設定:+斬れる第三陣営)を1人以上斬ってクルー陣営が勝利(タスク勝利除く)した時<br>

誰でも切れる。ベント不可。キルを起こす度に停電が起こる。(既に停電の場合はそのまま)。<br>

#### 設定

| 設定名                              |
| ----------------------------------- |
| キルクール                          |
| インポスター視界                   |
| キル可能第三陣営もカウントに含むか|

### ラブカッター

陣営 : 第三（単独）<br>
判定 : クルーメイト<br>
カウント : クルー<br>
勝利条件 : 設定回数誰かにキルされたら単独勝利。<br>

ラブカッターがキルされた時、キルをガードしてカウントする。
基本キル役職と同数回以下に設定するべきもの。

#### 設定

| 設定名                              |
| ----------------------------------- |
| 勝利条件のキルされる回数             |

### 純愛者

陣営 : 第三（単独）<br>
判定 : クルーメイト<br>
カウント : クルー<br>
勝利条件 : ラバーズ勝利と同様。<br>

誰かに向かってキルボタンを押すことで、その人と共にラバーズ(属性)になれる。ラバーズの仕様は通常ラバーズと全く同じ。<br>
設定時、ラバーズと共に入れることは出来ない。<br>

#### 設定

| 設定名                              |
| ----------------------------------- |
| 勝利を追加勝利にする             |


### ウォッチング
・属性<br>
匿名投票がオフになり、誰がどこに投票しているか分かるようになる。<br>

### ライティング
・属性<br>
視界が広くなる。設定で、停電中でも変わらない視界で見ることができる。<br>

| 設定名                              |
| ----------------------------------- |
| クルー視界増加率             |
| インポスター視界増加率             |
| 停電を無効にする             |

### サングラス
・属性<br>
視界が狭くなる。もし減少率により、0を下回ると見える部分がなく真っ暗になるので注意。<br>

| 設定名                              |
| ----------------------------------- |
| クルー視界減少率            |
| インポスター視界減少率            |

### シーイング
・属性<br>
プレイヤーが死ぬ時、キルフラッシュが見える。<br>
